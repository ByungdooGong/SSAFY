# Safefood 홈페이지 구현
- Spring MVC 구조로 구현
- DB : MySQL
- 소비자들에게 안전한 먹거리를 추천해주고 개인별로 알러지 정보를 기입하여 해당 식품이 안전하지 여부를 알려주는 홈페이지.
